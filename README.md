# 4-Bit Ripple Carry Adder

## Project Overview
A 4-Bit Ripple Carry Adder is a combinational circuit used to add two 4-bit binary numbers. It is constructed using four Full Adders connected in series.

## Components
- Full Adder
- Carry Input
- Carry Output
